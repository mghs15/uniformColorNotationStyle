# uniformColorNotationStyle
mapbox style specの色の表現方法を統一する

## 説明
* `uniform.js` : 色の記述方法を統一したスタイルを生成する。
* `convertColors.js` : 色の記述方法を統一したスタイルに対して、色をルールに基づいて変換する。
